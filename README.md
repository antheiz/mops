# Kumpulan Cerita MOP

Project ini diciptakan sebagai wadah untuk mengumpulkan dan menyebarluaskan "mop" — istilah lokal Papua untuk cerita lucu, *banyolan*, atau kisah ringan yang sering diceritakan dalam dialek Papua.
